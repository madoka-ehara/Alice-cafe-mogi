# pancake&cafe Alice(ぬい活応援カフェ)模擬企業サイト

## 概要
鳥取県デジタル人材育成プログラムの最終課題で、ぬい活を応援するカフェをテーマにしたホームページを制作しました

## 主な機能
- お知らせ
- 新着情報投稿
- 商品紹介
- 店舗情報
- アクセス（google mapの埋め込み）
- 営業日カレンダー（googleカレンダーの埋め込み）
- お問い合わせ
- SNS連携

## 開発環境
PHP 8.2.12  
MySQL  XAMPP MariaDB 10.4.32  
WordPress 6.9.1
